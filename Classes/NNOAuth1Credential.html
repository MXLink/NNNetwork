Properties

accessSecret

Access secret for credential.

@property (copy, readonly, nonatomic) NSString *accessSecret

Discussion

If you plan to use a credential with a NNOAuth1Client instance, this value should not be nil. If you do not have an access secret, this value should be an empty string.
