Class Methods

authorizationHeaderWithRequestMethod:requestURL:requestParameters:clientIdentifier:clientSecret:accessToken:accessSecret:signingMethod:privateKey:date:nonce:

Creates and returns an authorization header to be used in Authorization field of the HTTP request.

+ (NSString *)authorizationHeaderWithRequestMethod:(NSString *)requestMethod requestURL:(NSURL *)requestURL requestParameters:(NSDictionary *)requestParameters clientIdentifier:(NSString *)clientIdentifier clientSecret:(NSString *)clientSecret accessToken:(NSString *)accessToken accessSecret:(NSString *)accessSecret signingMethod:(NNOAuthSigningMethod)signingMethod privateKey:(NSData *)privateKey date:(NSDate *)date nonce:(NSString *)nonce

Parameters

requestMethod

Method of the OAuth request to be used in the header creation.

requestURL

OAuth request URL to be used in the header creation.

requestParameters

OAuth request parameters to be used in the header creation.

clientIdentifier

Client identifier for the OAuth service to be used in the header creation.

clientSecret

Client secret for the OAuth service to be used in the header creation.

accessToken

User access token for the OAuth service to be used in the header creation.

accessSecret

User access secret for the OAuth service to be used in the header creation.

signingMethod

Signing method to be used in the header creation. Defaults to HMAC-SHA1.

privateKey

The private key to be used for RSA-SHA1 signing method. If you are not using RSA-SHA1, set to nil.

date

Date to be used in the header creation.

nonce

Nonce to be used in the header creation.

Return Value

A new NSString object.
