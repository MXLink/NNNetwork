NSMutableURLRequest(NNNetwork) Category Reference

Declared in NSMutableURLRequest+NNNetwork.h

Overview

The NSMutableURLRequest(NNNetwork) category extends NSMutableURLRequest with a set of methods needed for networking.

Tasks

Signing URL Requests

  • – signForOAuth1WithClientIdentifier:clientSecret:accessToken:accessSecret:signingMethod:privateKey:parameters:

    Generates an OAuth 1.0 authorization header based on provided parameters. This method does not obtain URL parameters from existing request, but rather requires separate URL parameters to be passed in the parameters variable. This may be useful if you need to exclude some URL parameters from request signing.

Instance Methods

signForOAuth1WithClientIdentifier:clientSecret:accessToken:accessSecret:signingMethod:privateKey:parameters:

Generates an OAuth 1.0 authorization header based on provided parameters. This method does not obtain URL parameters from existing request, but rather requires separate URL parameters to be passed in the parameters variable. This may be useful if you need to exclude some URL parameters from request signing.

- (void)signForOAuth1WithClientIdentifier:(NSString *)clientIdentifier clientSecret:(NSString *)clientSecret accessToken:(NSString *)accessToken accessSecret:(NSString *)accessSecret signingMethod:(NNOAuthSigningMethod)signingMethod privateKey:(NSData *)privateKey parameters:(NSDictionary *)parameters

Parameters

clientIdentifier

The client identifier used in the signing process.

clientSecret

The client secret used in the signing process.

accessToken

The access token used in the signing process.

accessSecret

The access secret used in the signing process.

signingMethod

The OAuth signing method used in the signing process. May be HMAC-SHA1, RSA-SHA1 or plain text. Defaults to HMAC-SHA1.

privateKey

The private key to be used for RSA-SHA1 signing method. If you are not using RSA-SHA1, set to nil.

parameters

The parameters to be used in the signing process.
